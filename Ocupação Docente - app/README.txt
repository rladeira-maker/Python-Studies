Programa para preencher a ocupação docente

Para funcionar, o programa precisa ter atendidas algumas condições:
1) Deve haver os arquivos com os horários (qualquer nome), um horário
por aba;
2) Os horário devem ter um formato próprio e não podem ser alterado;
contudo, respeitadas as regras, pode ser ampliado o número de turmas
3) Na célula 'D2' de cada horário, em cada aba, deve conter o número de
aulas no horário (4, 5 ou 6 aulas por período).
4) O espaçamento entre às células 'SEG' (a partir de 'D3') deve ser man-
tido. No que seria a próxima célula 'SEG' (após a última turma), deve
ser colocado 'END'. Assim o programa entende que é a última turma.
4) Deve haver um arquivo com o nome GERAL.xlsx que contem todos os nomes
dos docentes e um formato próprio que não deve ser alterado
5) O arquivo Ocupacao_docentes.xslx não pode estar aberto
